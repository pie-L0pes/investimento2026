# 📱 Simulador de Investimentos

## 📌 Descrição do projeto

Este projeto é um aplicativo mobile desenvolvido em Dart utilizando o Flutter que permite simular o crescimento de um investimento ao longo do tempo.
O usuário pode inserir:

- Valor do investimento mensal
- Número de meses
- Taxa de juros mensal

A partir desses dados, o aplicativo calcula:

- Valor total acumulado sem juros
- Valor total acumulado com juros compostos

O objetivo é auxiliar na visualização do poder do investimento e do impacto dos juros compostos ao longo do tempo.
